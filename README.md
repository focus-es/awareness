# Awareness

Plataforma lúdica de entrenamiento perceptivo, atencional y de intuición.

**Fase 0** — Hub de entrada + 20 módulos cognitivos jugables + prácticas de visión intuitiva, publicados como sitio estático.

## Estructura

- `index.html` — hub de bienvenida ("¿Qué quieres hacer hoy?")
- `entrenamiento/` — 20 módulos cognitivos (bloques: atención, inhibición, percepción, memoria, visualización, intuición cognitiva, regulación) y 2 prototipos de sesión guiada
- `intuicion/` — El Orbe (visión intuitiva) y Visión remota

## Hoja de ruta

| Fase | Qué añade |
|------|-----------|
| 0 | Esta web: todo el material jugable online |
